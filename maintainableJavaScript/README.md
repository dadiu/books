《编写可维护的javascript》

第一章

- 缩进；
- 句尾封号； 
- 换行（1、假如正常缩进为2像素，则换行的缩进为1像素； 2、不要在运算符之前换行）； 
- 对于函数和方法命名来说，第一个单词为动词；

    
   动词     | 含义               
   -------- |-------------       
   can      | 函数返回一个布尔值  
   has      | 函数返回一个布尔值  
   is       | 函数返回一个布尔值 
   get      | 函数返回一个非布尔值 
   set      | 函数用来保存一个值  
   
- 常量用大写字母表示；
- 构造函数用大驼峰命名法，使用时前面一定会有new；
- NUMBER (1、不要省略“小数部分”或“整数部分”； 2、禁止使用八进制、十六进制、科学计数)；
